About App

This is a simple Angular App with the use of Bootstrap 5 UI Framework

App Features

Environmentally Friendly UI

The green theme feature is a way to make this app more environmentally friendly. It can be used to reduce the amount of energy used by this app.

Reactive Forms with Validators

Reactive forms are model-driven, which means that the form data is stored in an object that is bound to the form controls. This makes it easy to access and manipulate the form data. The user can validate the forms in real-time.

Edit Profile with Two-Way Buiding

User can edit profile details using two-way binding feature in this app.

Products List

This feature allows users to view a list of products. The user can view the single product by clicking on it.
